# surveyed

> A dope stepwise question component 

## Install

```bash
npm install --save surveyed
```

## Usage

```tsx
import React, { Component } from 'react'

import Surveyed from 'surveyed'
import 'surveyed/dist/index.css'

class Example extends Component {
  render() {
    return <Surveyed />
  }
}
```
